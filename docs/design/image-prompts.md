# UI Asset Prompts — כוכבניק

## Master Style Prompt
Premium cinematic festive evening stage for a Hebrew community survey game set during an Independence Day local event. Open-air celebration energy, broadcast polish, electric blue and white highlights, restrained soft gold accents, warm and inviting mood, elegant light beams, haze, bloom, particles, and depth. High-end, original, not stock, not cartoon, not childish, not purple cyberpunk, no embedded UI, no text, no logos, no people unless explicitly requested.

## Style Anchor
Primary visual anchor: `public/branding/hero-poster-anchor-v1.png`

Every future generated asset for the player-facing UI should treat this poster as the first reference for:
- painterly finish
- festive blue-white glow
- fireworks treatment
- warm community-event atmosphere
- family-friendly premium tone

Do not drift away from this reference into generic gradients, cyberpunk, flat vector visuals, or stock-photo realism.
